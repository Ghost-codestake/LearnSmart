Course Curriculum

1
Introduction to Cybersecurity
2 weeks

Fundamentals of cybersecurity, threat landscape overview, basic security principles, and introduction to risk management. You'll learn about different types of cyber threats, attack vectors, and the importance of cybersecurity in modern business.

2
Network Security
3 weeks

Network protocols, firewall configuration, intrusion detection systems, VPNs, and network monitoring. Hands-on experience with Wireshark, pfSense, and other network security tools.

3
Ethical Hacking & Penetration Testing
4 weeks

Learn ethical hacking methodologies, vulnerability assessment, penetration testing frameworks, and how to think like an attacker to better defend systems. Includes hands-on labs with Kali Linux, Metasploit, and Nmap.

4
Incident Response & Digital Forensics
3 weeks

Incident response procedures, digital forensics techniques, evidence collection and preservation, and malware analysis. Learn to investigate security incidents and recover from cyberattacks.

5
Security Governance & Compliance
2 weeks

Security frameworks (NIST, ISO 27001), compliance requirements (GDPR, HIPAA), risk assessment methodologies, and security policy development. Understanding the business side of cybersecurity.

6
Capstone Project
2 weeks

Apply all learned skills in a comprehensive capstone project. Design and implement a complete security solution for a simulated organization, including risk assessment, security architecture, and incident response plan.
